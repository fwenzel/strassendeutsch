Über {{ strassendeutsch }}

{{ strassendeutsch }} ist ein Wörterbuch der deutschen Umgangssprache.

Warum {{ strassendeutsch }}?

Ich wollte ein paar neue Technologien ausprobieren und brauchte ein einigermaßen nützliches Projekt dazu.

Hinzu kommt, dass die englische Umgangssprache ein hervorragendes Online-Wörterbuch namens Urban Dictionary besitzt. Für das Deutsche gibt es eine Handvoll solcher Webseiten, die allesamt ganz fürchterlich aussehen und alles andere als angenehm zu benutzen sind. Vielleicht kann mein Experiment dieses Problem beheben?

Lizenz

{{ strassendeutsch }} ist ein Software-Experiment von Fred Wenzel, das unter anderem die Technologien Flask, MongoDB und ElasticSearch verwendet. Diese Software ist Open Source und unter der GPL-Lizenz auf GitHub verfügbar. Pull-Requests willkommen!

Die Einträge in diesem Wörterbuch sind unter einer Creative-Commons-Lizenz (Namensnennung-Nicht-kommerziell-Weitergabe) verfügbar. Eine Kommerzielle Weiterverwendung ist nicht zulässig.
